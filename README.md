# 我的猜灯谜游戏

这是一个简单的猜灯谜小游戏，使用 HTML、CSS 和 JavaScript 实现。

## 功能

- 开始游戏：进入游戏界面，开始猜灯谜。
- 每日一言：提供每日一句励志名言。
- 创作者团队：展示游戏的创作团队信息。
- 小彩蛋：隐藏的小彩蛋功能。
